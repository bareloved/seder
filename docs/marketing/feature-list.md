## Suggested Grouping for Paywall Screen

If you ever add a free/pro split, here's how to group them:

### Free
- ניהול הכנסות במקום אחד
- מעקב סטטוס חשבוניות
- מעקב תשלומים
- עד 3 קטגוריות
- עברית מלאה
- פרטיות מלאה

### Pro
- Everything in Free, plus:
- ייבוא אוטומטי מיומן גוגל
- דשבורד חודשי עם KPIs
- אנליטיקס וגרפים
- ניהול לקוחות
- קטגוריות ללא הגבלה
- חישוב מע״מ אוטומטי
- תזכורות חכמות
- ייצוא נתונים
- אפליקציית אייפון + סנכרון
- התראות פוש
- כל העדכונים העתידיים
